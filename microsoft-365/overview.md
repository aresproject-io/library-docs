# Overview

## What is Microsoft 365

Microsoft 365 is a cloud-based subscription service that provides a suite of productivity tools and services for businesses and individuals. It includes the latest versions of the Microsoft Office suite (Word, Excel, PowerPoint, Outlook, and OneNote), as well as cloud-based versions of Microsoft's next-generation communications and collaboration services, such as Microsoft Teams, SharePoint Online, and Exchange Online.
